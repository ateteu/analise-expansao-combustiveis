# Estratégia de Expansão no Mercado de Combustíveis (Sudeste)

## Visão Geral

Este projeto simula um cenário real de negócio no qual uma distribuidora de combustíveis, com origem em Belo Horizonte (MG) e atuação consolidada em Minas Gerais, busca expandir suas operações no estado de São Paulo.

A análise utiliza dados públicos da ANP para entender o comportamento do consumo de combustíveis em nível municipal, com o objetivo de identificar mercados com maior potencial de demanda e crescimento.

O foco do projeto é transformar dados em insights que apoiem decisões estratégicas de expansão, considerando diferenças regionais, evolução temporal e perfil de consumo.

---

## Problema de Negócio

Quais municípios apresentam maior potencial para expansão, considerando:

- volume de consumo de combustíveis  
- crescimento ao longo do tempo  
- perfil de consumo por tipo de combustível  

---

## Fontes de Dados

Os dados foram obtidos a partir da Agência Nacional do Petróleo (ANP), via portal dados.gov.br.

Bases utilizadas:
- Vendas de gasolina C por município  
- Vendas de óleo diesel por município  

---

## Etapas do Projeto

- Preparação e padronização dos dados  
- Integração de múltiplas bases  
- Criação de métricas (crescimento, totais, participação)  
- Análise exploratória dos dados  
- Visualização dos resultados (Power BI)  

---

## Ferramentas Utilizadas

- SQL  
- Python (Pandas)  
- Power BI  

---

## Principais Análises

- Ranking de municípios por volume de consumo  
- Evolução temporal do consumo  
- Identificação de municípios com maior crescimento  
- Comparação entre tipos de combustível  
- Identificação de padrões regionais  

---

## Resultados Esperados

- Identificação de cidades com maior potencial de expansão  
- Compreensão do comportamento regional do consumo  
- Geração de recomendações estratégicas
